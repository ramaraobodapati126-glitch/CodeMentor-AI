# CodeMentor-AI


CodeMentor AI is an AI-powered coding mentor that helps developers learn, analyze, and improve their code through intelligent feedback and personalized insights.

## Features

* AI Chat Assistant for coding help
* Code Analysis with detailed feedback
* Dashboard with performance insights
* Mistake tracking and recommendations
* Strength and weakness analysis
* CodeLab for practice and testing

---

## Tech Stack

* Frontend: React (Vite)
* Styling: Tailwind CSS
* State Management: Context API
* Build Tool: Vite
* Linting: ESLint

---

## Getting Started

1. Clone the repository:

```
git clone https://github.com/yourusername/codementor-ai.git
```

2. Navigate to project:

```
cd codementor-ai
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```
